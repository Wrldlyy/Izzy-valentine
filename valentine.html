How to run locally

Save this file as valentine.html anywhere on your computer, then double-click it (or drag it into Chrome / Firefox). No server needed — it's a single self-contained HTML file.

How to host as a shareable link

GitHub Pages: Push the file into a GitHub repo, go to Settings → Pages, select the branch, and your link will be https://you.github.io/repo/valentine.html.

Netlify: Drag-and-drop the file onto app.netlify.app and you get an instant URL in seconds. Both are free.
